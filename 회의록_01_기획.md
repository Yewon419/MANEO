## 컨셉

데스크탑에 항상 떠 있는 투명 위젯 형태의 작은 생명체 Meum을 키우는 게임.
말은 못하지만 표정·동작·의성어로 감정을 표현하며, LLM이 유저 입력을 해석해 행동을 결정.
유저와의 상호작용이 쌓이면서 Meum은 조금씩 변해간다.

---

## 기술 스택

| 항목 | 내용 |
|------|------|
| 플랫폼 | 데스크탑 위젯 (Electron) |
| 언어 | HTML / CSS / JavaScript |
| 창 형태 | 프레임리스 + 투명 배경 + Always-on-top |
| LLM | Gemini 2.0 Flash (저비용, 무료 tier) |
| 배포 | .exe 빌드 |

---

## LLM 활용 방식

- 유저 채팅 입력 → LLM이 상황 해석 → 행동 지시 JSON 출력
- LLM은 말을 생성하는 게 아니라 **행동을 지시**하는 역할

```json
{
  "emotion": "confused",
  "sound": "끼액?",
  "action": "head_tilt"
}
```

---

## 메모리 시스템

- `memory.md` 파일에 자동 기록
- LLM이 매 대화 후 기억할 만한 내용이면 자동 업데이트
- 다음 대화 시 memory.md를 컨텍스트로 주입

```
memory.md 구조
├── [유저 정보] 이름, 말투 특징 등
├── [중요 사건] 날짜별 기록
├── [관계 기록] Meum이 유저에 대해 아는 것
└── [Meum 상태] 현재 기분, 좋아하는 것
```

---

## 성장 시스템

- **3단계** 성장 (시간 경과 + 상호작용 빈도)
- 성장 시각화: **팔다리 길이**로 표현

```
1단계 (유아)    2단계 (소년)    3단계 (청년)
뭉툭한 팔다리   중간 팔다리     긴 팔다리
행동 제한적     동작 늘어남     다양한 표현
```

- 3단계 이후: 유저와의 상호작용에 의해 **아주 가끔, 아주 미세하게** 변화
  - 자세, 습관, 반응 패턴 등

## 캐릭터 개성 시스템

탄생 시 랜덤 배정. 유저마다 다른 Meum.

### 눈 타입 (6종)
| # | 스타일 | 연결 성격 |
|---|--------|----------|
| 1 | 점 두 개 | 느긋함, 소식 |
| 2 | 반달 (아래로) | 게으름, 온화함 |
| 3 | 큰 동그라미 | 호기심, 활발함 |
| 4 | 비대칭 (크기 다름) | 제멋대로, 대담함 |
| 5 | 속 빈 동그라미 | 겁쟁이, 내향적 |
| 6 | 세로 타원 | 야성적, 예민함 |

### 재질 (5종)
| 재질 | 도트 표현 | 연결 성격 |
|------|----------|----------|
| 나무 | 결 라인, 따뜻한 갈색 | 느긋함, 고집있음 |
| 금속 | 하이라이트 점, 차가운 톤 | 무뚝뚝, 의외로 섬세 |
| 유리 | 반투명, 광반사 점 | 예민함, 겁쟁이 |
| 플라스틱 | 플랫 컬러, 단순 하이라이트 | 명랑함, 적응력 |
| 천 | 패턴 텍스처, 스티치 | 온화함, 애착형 |

### 색상
- 재질별 색상 팔레트 내에서 랜덤 배정

### 성격 파라미터 (탄생 시 결정)
- 겁쟁이 ↔ 대담함
- 게으름 ↔ 활발함
- 먹보 ↔ 소식
- 내향 ↔ 호기심 왕성
- 눈 타입 + 재질이 파라미터 가중치에 영향

## 비주얼 제작 방향

- **도구**: Leonardo AI 추천 (픽셀아트 전용 모델, 무료 tier 150크레딧/일)
- **모델**: AlbedoBase XL 또는 Pixel Art 스타일
- **비율**: 1:1 / 512×512

### 1차 테스트 프롬프트 (천 재질 + 점 눈 / 1단계)

**Positive:**
```
pixel art game sprite, tiny cute creature, round chubby body,
very short stubby arms and legs, two small dot eyes,
fabric texture body with visible stitch lines, soft pastel color,
idle standing pose, front facing,
transparent background, white outline,
simple minimal design, hollow knight style, ghibli spirit creature style,
no text, no background, single character
```

**Negative:**
```
background, realistic, 3d, anime, sharp edges,
complex details, text, watermark, multiple characters,
human, scary, ugly
```
