## 1. 구조

```
personality = {
  base:    { courage, energy, curiosity }   // 탄생 시 고정, 불변. 3축.
  drift:   { courage, energy, curiosity }   // Phase 3부터 유저 영향으로 이동
  current: base + drift                      // 실제 행동 판단 기준
}
```

- base: -1.0 ~ +1.0 (탄생 시 결정, 이후 변경 불가)
- drift: -0.3 ~ +0.3 (Phase 3 이전은 0, 이후 서서히 이동)
- current: clamp(base + drift, -1.0, +1.0)
- ~~appetite~~: 제거됨. 먹이 시스템 없음.

## 2. 네 축의 정의

### courage (겁쟁이 ↔ 대담)

**이 축이 결정하는 것: 새로운 자극에 대한 첫 반응**

| 구간 | 수치 | 행동 표현 |
|------|------|-----------|
| 겁쟁이 | -1.0 ~ -0.5 | 클릭 시 움츠러듦(shrink). 커서가 빠르게 접근하면 떨기(tremble). 새로운 상황에서 뒷걸음. 쓰다듬기 시 처음에 긴장했다가 서서히 이완. |
| 소심 | -0.5 ~ -0.1 | 클릭 시 살짝 놀람. 커서 접근에 경계하지만 도망은 안 감. 낯선 입력에 "끼릿?" 반응. |
| 보통 | -0.1 ~ +0.1 | 클릭에 중립 반응. 커서에 관심은 보이지만 적극적이진 않음. |
| 당찬 | +0.1 ~ +0.5 | 클릭 시 bounce로 반응. 커서를 피하지 않음. 탈출 이벤트에 적극적. |
| 대담 | +0.5 ~ +1.0 | 클릭에도 꿈쩍 안 함. 커서를 향해 다가감. 탈출 시 더 멀리 나감. 가끔 유저를 정면으로 노려봄(vertical_oval 눈). |

**idle 행동:**
- courage < -0.3: 위젯 구석에 웅크리는 경향. 가장자리에 가까이 안 감.
- courage > +0.3: 위젯 가장자리에서 삐꼼거리는 빈도 높음. 당당히 중앙에 서 있음.

**쓰다듬기 반응 차등:**
- courage < -0.5: 첫 터치에 flinch → 1초 후 서서히 purr. 신뢰를 얻어가는 느낌.
- courage > +0.5: 첫 터치부터 바로 purr. 머리를 들이밈.

---

### energy (게으름 ↔ 활발)

**이 축이 결정하는 것: 움직임의 빈도와 속도**

| 구간 | 수치 | 행동 표현 |
|------|------|-----------|
| 게으름 | -1.0 ~ -0.5 | 대부분 앉아있거나 누워있음. 움직이더라도 느릿느릿. 반응 딜레이 +0.5초. 하품 빈도 높음. 먹이 반응조차 느긋. |
| 느긋 | -0.5 ~ -0.1 | 앉아있는 시간이 서있는 시간보다 김. 움직임 부드럽고 천천히. 급한 게 없는 느낌. |
| 보통 | -0.1 ~ +0.1 | 서기/앉기 비율 5:5. 보통 속도. |
| 활발 | +0.1 ~ +0.5 | 서있는 시간이 긴다. 자주 움직임. 반응 빠름. 가끔 제자리 점프. |
| 과활동 | +0.5 ~ +1.0 | 거의 항상 움직임. 통통 뛰어다님. 반응 즉각. idle 상태에서도 제자리 탕탕. 잠들기까지 오래 걸림(energy 감소 속도 -20%). |

**idle 행동:**
- energy < -0.3: idle 상태의 80%가 sit/sleep 포즈. 움직이는 idle은 느린 wobble만.
- energy > +0.3: idle 상태에서도 bounce, walk, 제자리 탕탕. 가만히 있는 시간이 짧음.

**수면 진입 차등:**
- energy축 < -0.5: state.energy 40 이하에서 이미 졸려함. 빨리 잠듦.
- energy축 > +0.5: state.energy 15 이하까지 버팀. 졸리면서도 놀려고 함.

---

### curiosity (내향 ↔ 호기심)

**이 축이 결정하는 것: 환경과 유저 입력에 대한 탐색 욕구**

| 구간 | 수치 | 행동 표현 |
|------|------|-----------|
| 내향 | -1.0 ~ -0.5 | 위젯 중앙에서 거의 안 움직임. 새로운 입력에 느린 반응. unknown intent에도 무덤덤("..."). 자기 세계에 빠져있는 느낌. |
| 조용 | -0.5 ~ -0.1 | 가끔 주변을 보지만 적극적이진 않음. 반응은 하되 호들갑은 없음. |
| 보통 | -0.1 ~ +0.1 | 적당한 호기심. 새 입력에 head_tilt 반응. |
| 호기심 | +0.1 ~ +0.5 | 커서 움직임을 눈으로 추적. unknown intent에 "??" + big_circle 눈. 탈출 이벤트 빈도 증가. |
| 탐험가 | +0.5 ~ +1.0 | 커서를 적극 추적. 위젯 가장자리를 자주 탐색. unknown intent에 excited 반응. 탈출 시 두리번거리는 연출. 새로운 것을 발견한 듯한 표정(big_circle → 반짝). |

**idle 행동:**
- curiosity < -0.3: 한 자리에 오래 머묾. 눈 움직임도 적음. 명상하는 느낌.
- curiosity > +0.3: 위젯 안에서 이리저리 돌아다님. 눈이 자주 움직임. 가장자리를 기웃거림.

**커서 추적 차등:**
- curiosity < -0.3: 커서를 무시. 눈이 커서를 따라가지 않음.
- curiosity > +0.3: 눈이 커서를 따라감(시선 추적). bond 70+ 시 몸까지 커서 방향으로.

## 3. 축 간 조합 — 성격 아키타입

4축의 조합이 만드는 대표적 성격 유형들. 이건 시스템이 자동으로 만들어내는 것이지
코드로 하드코딩하는 게 아님. 참고용.

| 아키타입 | courage | energy | curiosity | 느낌 |
|----------|---------|--------|-----------|------|
| **느긋한 관찰자** | +0.1 | -0.6 | +0.7 | 느릿느릿 돌아다니면서 이것저것 구경. 급하지 않지만 호기심은 많음. 커서를 천천히 눈으로 따라감. |
| **대담한 돌진러** | +0.8 | +0.9 | +0.4 | 쉴 새 없이 뛰어다니고, 커서를 향해 돌진하고, 위젯 밖으로 자주 탈출. 에너지 넘침. |
| **내향적 겁쟁이** | -0.8 | -0.4 | -0.7 | 거의 안 움직이고, 건드리면 움츠러들고, 혼자 있는 걸 좋아함. bond가 올라가면 유저 앞에서만 살짝 긴장을 풂. |
| **쿨한 독립형** | +0.3 | -0.1 | -0.5 | 자기 페이스 유지. 주변에 관심 없어 보이지만 가끔 유저를 한번 쳐다보는 눈빛이 따뜻함. |
| **사교적 활발이** | -0.1 | +0.7 | +0.5 | 유저를 보면 반기고, 돌아다니면서 관심 끌기. 커서 따라가기 좋아함. 전형적인 강아지 성격. |
| **조용한 겁쟁이 탐험가** | -0.6 | -0.3 | +0.6 | 겁이 많지만 호기심을 못 이김. 몰래 가장자리까지 갔다가 유저가 보면 화들짝. 갭 모에. |

## 4. 재질 × 눈 → 성격 편향

재질과 눈 타입은 성격의 "편향"을 만든다. 100% 결정하는 게 아니라 방향을 틀어줌.

### 4.1 재질별 성격 컨셉

| 재질 | 컨셉 | courage | energy | curiosity |
|------|------|---------|--------|-----------|
| **wood** | 느긋한 고집쟁이 | 0.0 | -0.2 | -0.1 |
| **metal** | 무뚝뚝하지만 섬세 | +0.1 | 0.0 | -0.2 |
| **pearl** | 예민한 감수성 | -0.2 | -0.1 | +0.1 |
| **light** | 호기심 많은 겁쟁이 | -0.1 | +0.2 | +0.3 |
| **doll** | 온화한 애착형 | -0.1 | 0.0 | +0.1 |
| **robot** | 대담한 탐험가 (레어) | +0.3 | +0.3 | +0.4 |

### 4.2 눈 타입별 성격 컨셉

| 눈 | 컨셉 | courage | energy | curiosity |
|----|------|---------|--------|-----------|
| **dot** (점) | 느긋하고 무심 | 0.0 | -0.3 | -0.2 |
| **half_moon** (반달) | 게으르고 온화 | +0.1 | -0.4 | 0.0 |
| **big_circle** (큰원) | 호기심 왕, 활발 | 0.0 | +0.2 | +0.4 |
| **asymmetric** (비대칭) | 제멋대로, 대담 | +0.4 | +0.1 | +0.2 |
| **empty_circle** (속빈원) | 겁쟁이, 내향 | -0.4 | 0.0 | -0.2 |
| **vertical_oval** (세로타원) | 야성적, 예민 | +0.2 | +0.3 | 0.0 |

### 4.3 최종 성격 계산

```javascript
function generatePersonality(material, eyeType) {
  const axes = ['courage', 'energy', 'curiosity'];
  const base = {};
  
  for (const axis of axes) {
    const random   = (Math.random() * 0.6) - 0.3;   // -0.3 ~ +0.3
    const matBias  = MATERIAL_WEIGHTS[material][axis]; // §4.1
    const eyeBias  = EYE_WEIGHTS[eyeType][axis];       // §4.2
    base[axis] = clamp(random + matBias + eyeBias, -1.0, +1.0);
  }
  
  return {
    base,
    drift: { courage: 0, energy: 0, curiosity: 0 },
    get current() {
      return Object.fromEntries(
        axes.map(a => [a, clamp(this.base[a] + this.drift[a], -1.0, +1.0)])
      );
    }
  };
}
```

## 5. 행동 결정 흐름

성격이 화면에 보이는 전체 흐름:

```
[입력 발생]
    │
    ├─ 채팅 → intent 분류 → response_table 조회
    │                           │
    │                    성격 current로 반응 분기
    │                    (§2의 구간별 행동)
    │
    ├─ 쓰다듬기 → courage 기반 첫 반응 → mood 기반 이완
    │
    ├─ 클릭 → courage + energy 기반 반응
    │
    └─ idle (입력 없음) → idle_scheduler
                            │
                    ┌───────┴───────────────────┐
                    │  30초~2분 간격 랜덤 발동   │
                    │                            │
                    │  행동 풀에서 선택:          │
                    │  - energy축 → 움직임 빈도  │
                    │  - curiosity축 → 탐색 행동 │
                    │  - courage축 → 탈출 확률   │
                    │  - mood → 전체 톤 조정     │
                    │  - state.energy → 졸림 체크│
                    └────────────────────────────┘
```

## 6. Phase 3 drift 시스템 (엔드 컨텐츠)

> Phase 3 진입 전에는 drift = 0. 모든 행동은 base로만 결정.

### 6.1 drift 발동 조건

유저의 최근 N일(기본 7일) 상호작용 패턴을 추적:

```javascript
const patterns = {
  nightOwl:     // 22시~04시 상호작용 비율 > 40%
  comforter:    // comfort intent 비율 > 25%
  playful:      // play intent 비율 > 25%
  talkative:    // 일 평균 채팅 > 20회
  quiet:        // 일 평균 채팅 < 5회
  touchy:       // 쓰다듬기/클릭 비율 > 채팅 비율
};
```

### 6.2 패턴 → drift 매핑

| 유저 패턴 | drift 방향 | 크기/일 | 최대 |
|-----------|-----------|---------|------|
| nightOwl | energy +방향 | +0.005 | +0.15 |
| comforter | courage +방향 (겁쟁이가 점점 안심) | +0.005 | +0.2 |
| playful | energy +방향, curiosity +방향 | +0.003 | +0.15 |
| talkative | curiosity +방향 | +0.003 | +0.15 |
| quiet | curiosity -방향 (내향적으로) | +0.003 | +0.15 |
| touchy | courage +방향 (접촉에 익숙해짐) | +0.005 | +0.2 |

- drift는 하루 1회, 자정 기준으로 갱신
- 패턴이 사라지면 drift는 유지 (감소 안 함) — "한번 변한 성격은 돌아가지 않는다"
- 최대 ±0.3 캡

### 6.3 drift의 체감

drift가 0.1만 움직여도 행동이 달라지는 게 보여야 함:
- 예: courage base=-0.7(겁쟁이)인데 touchy 패턴으로 drift +0.2
  → current=-0.5 → 여전히 소심하지만, 쓰다듬기에 flinch 안 하고 바로 purr
  → "이 애가 나한테 마음을 열었다"는 느낌

이 변화를 유저에게 명시적으로 알리지 않음. 설계 원칙 참조(game_identity.md §5).
